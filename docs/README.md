## 구현 기능 목록
### 로또 번호 자동 생성기
- [ ] 1 ~ 45 중 중복되지 않도록 6 숫자 뽑기
- [ ] 1 ~ 45 중 앞에서 뽑은 6개 숫자와 중복되지 않도록 보너스 번호 1개 뽑기
### 로또 판매자
- [ ] 구매 금액 받기
- [ ] 구매 금액만큼 로또 발급 : 발급 개수 반환
  - [ ] 오름차순으로 정렬
### 로또 구매자
- [ ] 구매 금액 지불
  - 예외 : 1,000 원 단위로 지불하지 않은 경우
  - 예외 : 숫자가 아닐 경우 예외
### 로또 번호 지정자
- [ ] 6자리 당첨 번호 입력받기
  - 예외 : 6 자리가 아닐 경우
  - 예외 : 숫자가 아닐 경우
  - 예외 : 중복 숫자 있을 경우
  - 예외 : 1 ~ 45 범위를 벗어날 경우
- [ ] 보너스 번호 입력받기
  - 예외 : 숫자가 아닐 경우
  - 예외 : 중복 숫자 있을 경우
  - 예외 : 1 ~ 45 범위를 벗어날 경우
### 로또 결과 판정자
- [ ] 구매한 로또 번호와 정답 로또 비교하여 등수 매기기 
  - 당첨 결과 Enum
### 당첨 결과 Enum
- [ ] 1등, 2등, 3등, 4등, 5등
### 수익률 계산기
- [ ] 수익율 구해서 반환
